## Instruções

- Clone este repositório pelo terminal e builde/rode o projeto usando o gcc:

```
git clone https://github.com/guilarre/projeto_pe.git
cd projeto_pe
gcc programinho.c -o programinho
./programinho
```
- Digite a opção desejada no menu e siga as instruções de uso na tela

## Descrição

O sistema lê o arquivo produtos.txt e armazena-os na memória dinamicamente (o tamanho do array que armazena os produtos é incrementado conforme necessário)

O sistema permite:
- Adicionar novos produtos;
- Buscar produto por código;
- Imprimir todos os produtos em memória;
- Ordenar os produtos por preço e imprimir;
